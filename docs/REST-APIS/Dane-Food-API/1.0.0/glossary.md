# Glossary

This glossary defines the foundational technical terms, logistics concepts, and data metrics utilized across the Dane Food API ecosystem.

---

* **Asynchronous Webhook:** A user-defined HTTP callback path that triggers automatically to notify an external system when a background processing event finishes executing.
* **Bearer Token:** A security token that grants access to a protected resource to any party that possesses the token string, typically transmitted inside an HTTP Authorization header.
* **Distribution Center:** A dedicated logistics warehouse node responsible for tracking localized product reserves and handling physical order fulfillment routing.
* **JSON Web Token (JWT):** A compact, URL-safe means of representing claims to be transferred between two parties, used here to securely transmit authenticated user identity attributes.
* **Quantity on Hand (QOH):** The real-time metric tracking the exact physical inventory volume of a specific product brand currently sitting on the shelves of a distribution hub.
* **Universal Product Code (UPC):** A standardized barcode string used to isolate and verify specific product inventory lines across distinct brand types.
