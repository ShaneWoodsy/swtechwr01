# Services

## Docs as Code Services

I provide comprehensive technical writing and documentation services using modern docs-as-code practices.

### Core Services

#### Documentation Strategy & Planning
- Audit existing documentation and identify gaps
- Develop documentation roadmaps aligned with product development
- Create information architecture for complex documentation sets
- Establish documentation best practices and workflows

#### API Documentation
- Create and maintain OpenAPI specifications
- Write API reference documentation
- Develop API integration guides and tutorials
- Produce SDKs and code sample documentation

#### User Documentation
- Write user guides and tutorials
- Create quick start guides and getting started documentation
- Develop troubleshooting and FAQs
- Produce release notes and changelog documentation

#### Style Guides & Standards
- Develop internal style guides and voice guidelines
- Create documentation templates and patterns
- Establish terminology databases and glossaries
- Build reusable content libraries

#### Technical Implementation
- Documentation repository setup and maintenance
- Static site generator configuration (Hugo, Jekyll, Sphinx, etc.)
- CI/CD pipeline integration for documentation
- Documentation versioning and deployment

### Tools & Technologies

**Documentation Formats**: Markdown, reStructuredText, AsciiDoc, OpenAPI/Swagger  
**Version Control**: Git, GitHub, GitLab  
**Static Site Generators**: Hugo, Jekyll, Sphinx, MkDocs  
**API Documentation**: OpenAPI 3.0, Postman, API Blueprint  
**Collaboration**: GitHub, Jira, Confluence  
**Development Tools**: VSCode, IntelliJ IDEA  

### Engagement Models

- **Retainer**: Ongoing documentation support and maintenance
- **Project-based**: Complete documentation projects from planning through deployment
- **Contract**: Full-time W2 employment opportunities
- **Consulting**: Strategic documentation assessment and recommendations
